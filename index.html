Glosario de etiquetas HTML

  • <head>Rrovee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos. </head>
  • <body> Representa el contenido de un documento HTML </body>
  • <p> Muestra un parrafo para textos concretos o textos normales </p>
  • <br> Salto de linea
  • <hr> Representa un salto temático a nivel de párrafo.
  • <h1-h6> Representa titulos y subtitulos
  • <nav> Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos.
  • <header> Representa un grupo de ayudas introductorias o de navegación
  • <main> Representa el contenido principal del <body> de un documento o aplicación.
  • <footer> Contiene información acerca de el autor de la sección, datos de derechos de autor o enlaces a documentos relacionados
  • <section> Representa una sección genérica independiente de un documento, que no tiene un elemento semántico más específico para representarla. Las secciones siempre deben tener un título, con muy pocas excepciones.
  • <article> Representa una composición auto-contenida en un documento, una página, una aplicación o en un sitio, que se quiere que sea distribuíble y/o reutilizable de manera independiente
  • <div> Define las divisiones lógicas existentes en el contenido de una página web.
  • <aside> Es un elemento semántico que se introduce para marcar el contenido que está indirectamente relacionado con el contenido principal del artículo de una página.
